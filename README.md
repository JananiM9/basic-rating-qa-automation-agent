# Basic Rating QA Automation Agent

A simple VS Code extension that hosts an AI prompt library for the QA team.

The prompts help build a basic Python + Selenium + pytest-bdd test automation framework.

## Try it locally

1. Open this folder in VS Code.
2. Press `F5` — a new VS Code window opens with the extension loaded.
3. In that new window, press `Ctrl+Shift+P` and run **Basic QA: Hello**.
4. You should see a notification confirming the extension is active.
